## 关于 Linux 用户协会

Linux 用户协会是由在校的 GNU/Linux 爱好者发起并组成的一个全校性群众团体。成立协会的目的在于联合学校的 GNU/Linux 使用者，搭建信息交流共享的平台，宣传自由软件的价值，提高自由软件社区文化氛围，推广自由软件在校园乃至当地地区的应用。
